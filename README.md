# JavaScript Array Methods & Callbacks ⚡

This repository contains the fourth homework assignment for the GoIT Fullstack Developer course. It focuses on functional programming patterns in JavaScript, specifically **callbacks**, **closures**, and powerful **array iteration methods**.

🔗 **[Live Demo](https://ibuizle.github.io/goit-js-hw-04/)**

## 📖 About the Project

The goal of this assignment was to move away from traditional `for` and `while` loops and master the declarative approach to data manipulation. The tasks involve processing complex data structures using methods that make the code more readable and maintainable.

### Key Learnings & Features:
- **Callback Functions:** Learning how to pass functions as arguments to other functions to create flexible and reusable logic.
- **Array Methods (map, filter, find):** Transforming and filtering data sets without mutating the original arrays.
- **Sorting & Reducing (sort, reduce):** Implementing complex logic to calculate totals or sort data based on multiple criteria.
- **Arrow Functions:** Using concise ES6 syntax for writing cleaner callback logic.
- **Lexical Environment & Closures:** Understanding how functions remember the scope in which they were created.



## 🛠 Built With

- **HTML5:** Basic structure for testing logic.
- **Vanilla JavaScript:** ES6+ functional programming features.
